# FthnLabs Display

An Arduino libary for driving dot matrix display. Currently, only support ESP32.
This library also supports AdafruitGFX, so display manipulation can be easily done
by using AdafruitGFX functionalities.

## Supported Display

1. HUB12 dot matrix panel.
